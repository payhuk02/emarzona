## Introduction

Emarzona est une plateforme e-commerce complète qui vous permet de vendre différents types de produits :

- **Produits digitaux** : Logiciels, ebooks, templates, etc.
- **Produits physiques** : Vêtements, accessoires, objets, etc.
- **Services** : Consultations, formations, réservations, etc.
- **Cours en ligne** : Formations structurées avec leçons, quiz, certificats
- **Œuvres d'artiste** : Art, musique, livres, designs, etc.

## Création de compte

1. Cliquez sur "S'inscrire" en haut à droite
2. Remplissez le formulaire avec vos informations
3. Vérifiez votre email
4. Connectez-vous à votre compte

## Gestion de boutique

### Créer une boutique

1. Allez dans "Mes boutiques" > "Créer une boutique"
2. Remplissez les informations :
   - Nom de la boutique
   - Slug (URL unique)
   - Description
   - Logo et bannière
3. Configurez les paramètres de paiement
4. Activez votre boutique

### Paramètres de boutique

- **Informations générales** : Nom, description, contact
- **Paiements** : Intégration Moneroo/PayDunya
- **Expédition** : Zones, transporteurs, tarifs
- **SEO** : Meta tags, descriptions optimisées

## Produits

### Créer un produit digital

1. Allez dans "Mes produits" > "Créer un produit"
2. Sélectionnez "Produit digital"
3. Remplissez les informations :
   - Nom, description, prix
   - Fichiers à télécharger
   - Licence (simple, multi, illimitée)
   - Mises à jour automatiques
4. Configurez les variantes si nécessaire
5. Publiez le produit

### Créer un produit physique

1. Sélectionnez "Produit physique"
2. Informations de base :
   - Nom, description, prix
   - SKU, poids, dimensions
3. Variantes :
   - Couleurs, tailles, matériaux
   - Prix et stock par variante
4. Expédition :
   - Classes d'expédition
   - Zones et tarifs
5. Inventaire :
   - Stock initial
   - Alertes de stock bas
6. Publiez le produit

### Créer un service

1. Sélectionnez "Service"
2. Informations :
   - Nom, description, durée
   - Disponibilité (calendrier)
   - Staff assigné
   - Ressources nécessaires
3. Réservations :
   - Créneaux disponibles
   - Politique d'annulation
4. Publiez le service

### Créer un cours en ligne

1. Sélectionnez "Cours en ligne"
2. Informations :
   - Nom, description
   - Sections et leçons
   - Quiz et évaluations
   - Certificat de complétion
3. Contenu :
   - Vidéos, documents, ressources
   - Progression automatique
4. Publiez le cours

### Créer une œuvre d'artiste

1. Sélectionnez "Œuvre d'artiste"
2. Informations :
   - Type d'artiste (écrivain, musicien, artiste visuel, etc.)
   - Titre de l'œuvre
   - Année, médium, dimensions
   - Édition (originale, limitée, reproduction)
3. Certificat d'authenticité
4. Collections (optionnel)
5. Publiez l'œuvre

## Commandes

### Gérer les commandes

1. Allez dans "Commandes"
2. Filtrez par statut, date, produit
3. Actions disponibles :
   - Voir les détails
   - Mettre à jour le statut
   - Générer une étiquette d'expédition
   - Traiter un remboursement

### Expédition

1. Sélectionnez une commande
2. Cliquez sur "Générer étiquette"
3. Choisissez le transporteur
4. Imprimez l'étiquette
5. Marquez comme expédié

### Retours

1. Un client demande un retour
2. Vérifiez la demande
3. Approuvez ou rejetez
4. Si approuvé, générez une étiquette de retour
5. Traitez le remboursement

## Paiements

### Configuration

1. Allez dans "Paramètres" > "Paiements"
2. Configurez Moneroo ou PayDunya
3. Testez les transactions
4. Activez en production

### Transactions

- Voir toutes les transactions
- Filtrer par statut, date, montant
- Exporter les rapports
